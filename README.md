# L'Dor Vador — Jewish Heritage Travel to Curaçao

A Tree Crown Studio client project.

## What this is

A single self-contained homepage mockup to approve the **look, structure, and new sections**
before we rebuild the real site on Squarespace. Everything is placeholder-realistic — trip
names, dates, prices, and testimonials get swapped for Hannah's real content.

### What it demonstrates
- Cobalt-led brand identity derived from the existing logo (`#1150AB`) and the colours of Curaçao —
  the ochre synagogue walls and the terracotta Handelskade waterfront.
- Elderly-first legibility: large type, big tap targets, high contrast, gentle motion.
- A signature "zoom into the island" map moment with a pin on Willemstad.
- New sections: **upcoming trips with live availability**, services, testimonials, an FAQ,
  and a real inquiry form (replacing the current mailto link).
- Live **English / Spanish / Dutch / Hebrew** language switch, including right-to-left for Hebrew
  (a preview of the planned Weglot translation on Squarespace).

## Not yet real (prototype vs. final build)
- This is hand-coded to approve the direction. The production site will be rebuilt natively on
  **Squarespace** so the owner can edit it easily.
- Trip availability will run on Squarespace commerce/events; the four languages via **Weglot**
  (a paid add-on, ~€15–29/mo — to confirm with the client).
- Photography here is pulled from the current live site; final assets and the vector logo to come.

## Enabling the preview link
1. Repo **Settings → Pages**
2. **Source:** Deploy from a branch → **Branch:** `main` → **/ (root)** → **Save**
3. Wait ~1 minute; the link above goes live.

The page is set to `noindex` so it won't appear in search while it's a private preview.
